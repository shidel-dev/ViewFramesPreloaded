ViewFrames is a lightweight JavaScript framework, which treats iframes as a first class citizen of the web. ViewFrames allows for an iframe based MVC pattern, which facilitates painless and preformant one page applications.

In addition, viewFrames fixes many of the problems of iframes by matching the features provided by the ‘seamless’ iframe attribute, which has failed to gain traction on critical browsers.

Why viewFrames?

  • Iframes are sandboxed
  • Quickly develop your controller and views
  • Avoid conflicts with parent page
  • Good browser support

Features

Manage your page's history

Push history events when changing views, and for diffrent application states.

Define routes for your views

ViewFrames has a built in routing system which handles page actions, and events. Routes can be specified or implied.

Makes the iframe seemless

ViewFrames matches many of the features preposed by the seemless iframe attribute. This allows you to create iframes which your users will never even know are there.
